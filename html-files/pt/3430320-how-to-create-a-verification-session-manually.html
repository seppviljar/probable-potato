Como criar uma sessão de verificação manualmente

Às vezes você precisa criar sessões manualmente - aqui está como fazer isso

A Veriff Station oferece a opção de criar sessões de verificação com um simples toque de botão. Este artigo explica exatamente como fazer isso!

  1. Faça login na sua conta Veriff Station

  2. Vá para a aba Verificações

  3. Clique em ADICIONAR VERIFICAÇÃO

A seguinte tela aparecerá:

Primeiramente, você deve selecionar qual integração será utilizada para criar sessões de verificação.

NOTA: As sessões de verificação serão processadas pela Veriff apenas em integrações ao vivo, então, se você estiver criando uma sessão enquanto testa ou em um ambiente de produção - certifique-se de que você selecione uma integração ao vivo. Para entender as diferenças entre uma integração ao vivo e uma integração de teste, consulte este artigo.
​
Após selecionar uma integração, você tem a opção de preencher o primeiro e o sobrenome da pessoa para quem você está criando a sessão, no entanto, isso é completamente opcional, pois a Veriff irá extrair o primeiro e o sobrenome das imagens do documento enviado e retorná-los para você via API ou através da interface do usuário da Veriff Station.

Você também pode adicionar um ID único, no entanto, isso também é completamente opcional. Cabe a você decidir o que usar como identificador único - pode ser o ID de usuário do seu sistema, endereço de e-mail, número do pedido, número de telefone ou qualquer outro ID único que você escolher.
​
Clique em GERAR VERIFICAÇÃO e você verá a seguinte tela

Agora você tem duas opções para iniciar a sessão de verificação - você pode entrar no fluxo de verificação escaneando o código QR com sua câmera OU acessando o link diretamente.

Você também pode compartilhar ambas as opções clicando no botão de compartilhamento e encaminhando um dos métodos mencionados a uma pessoa que você deseja verificar.
